C++

Мощь и контроль

C++ - это компилируемый язык общего назначения, который даёт программисту полный контроль над системными ресурсами и памятью. Он сочетает в себе возможности языков высокого и низкого уровней.

Ключевые особенности:

  • Объектно-ориентированное программирование
  • Шаблоны (generics)
  • Прямая работа с памятью (указатели)
  • Мультипарадигменность
  • Высокая производительность

Где применяется:

  • Системное программирование (ОС, драйверы)
  • Разработка игр (Unreal Engine)
  • Высокопроизводительные вычисления
  • Встраиваемые системы
  • Финансовые системы

Пример кода: Калькулятор

#include <iostream>
using namespace std;

int main() {
    double num1, num2;
    char operation;

    cout << "Введите первое число: ";
    cin >> num1;
    cout << "Введите операцию (+, -, *, /): ";
    cin >> operation;
    cout << "Введите второе число: ";
    cin >> num2;

    switch(operation) {
        case '+':
            cout << "Результат: " << num1 + num2;
            break;
        case '-':
            cout << "Результат: " << num1 - num2;
            break;
        case '*':
            cout << "Результат: " << num1 * num2;
            break;
        case '/':
            if (num2 != 0)
                cout << "Результат: " << num1 / num2;
            else
                cout << "Ошибка: деление на ноль!";
            break;
        default:
            cout << "Неподдерживаемая операция!";
    }

    return 0;
}

Python

Простота и универсальность

Python - интерпретируемый язык высокого уровня с динамической типизацией. Его философия подчёркивает простоту и читаемость кода, что делает его идеальным для обучения и быстрой разработки.

Ключевые особенности:

  • Простой и понятный синтаксис
  • Динамическая типизация
  • Автоматическое управление памятью
  • Большая стандартная библиотека
  • Кроссплатформенность

Где применяется:

  • Веб-разработка (Django, Flask)
  • Анализ данных и машинное обучение
  • Автоматизация и скриптинг
  • Научные вычисления
  • Разработка прототипов

Пример кода: Генератор чисел Фибоначчи

def fibonacci(n):
    """Генерирует последовательность Фибоначчи"""
    a, b = 0, 1
    sequence = []
    for _ in range(n):
        sequence.append(a)
        a, b = b, a + b
    return sequence

# Получаем ввод от пользователя
try:
    n = int(input("Введите количество чисел Фибоначчи: "))
    if n <= 0:
        print("Пожалуйста, введите положительное число.")
    else:
        fib_sequence = fibonacci(n)
        print(f"Последовательность Фибоначчи из {n} чисел:")
        print(fib_sequence)
except ValueError:
    print("Ошибка: пожалуйста, введите целое число.")

HTML & CSS

Основы веб-разработки

HTML (HyperText Markup Language) и CSS (Cascading Style Sheets) - фундаментальные технологии для создания веб-страниц. HTML определяет структуру контента, а CSS отвечает за его визуальное представление.

Ключевые особенности HTML:

  • Семантическая разметка контента
  • Иерархическая структура документа
  • Поддержка мультимедиа
  • Формы для ввода данных
  • Доступность (accessibility)

Ключевые особенности CSS:

  • Селекторы для выбора элементов
  • Каскадность и наследование
  • Адаптивный дизайн (медиазапросы)
  • Анимации и переходы
  • Flexbox и Grid для макетов

Где применяется:

  • Создание веб-страниц и сайтов
  • Веб-приложения
  • Электронные письма
  • Документация
  • Визуализация данных

Какой язык выбрать?

Выбор языка зависит от ваших целей:

  • C++ - для системного программирования, высокопроизводительных приложений
  • Python - для быстрой разработки, анализа данных, машинного обучения
