Многослойная сеть

Нейроны объединяются в слои: входной, скрытые, выходной

Входной слой

Получает исходные данные

Скрытые слои

Выполняют преобразования данных

Выходной слой

Выдает результат работы сети

Типы нейронных сетей

Специализированные архитектуры для разных задач

Сверточные сети (CNN)

Для работы с изображениями и видео

  • Сверточные слои для выявления признаков
  • Пулинг-слои для уменьшения размерности
  • Применение: классификация изображений, распознавание объектов

Рекуррентные сети (RNN/LSTM)

Для последовательных данных (текст, речь, временные ряды)

  • Память для учета предыдущих состояний
  • LSTM и GRU для решения проблемы затухающего градиента
  • Применение: машинный перевод, генерация текста

Трансформеры

Для обработки естественного языка (NLP)

  • Механизм внимания (attention)
  • Параллельная обработка последовательностей
  • Применение: GPT, BERT, современные LLM

GAN (Generative Adversarial Networks)

Для генерации новых данных

  • Две сети: генератор и дискриминатор
  • Состязательное обучение
  • Применение: создание изображений, анимаций, стилизация

import tensorflow as tf
from tensorflow.keras import layers, models

# Создаем модель
model = models.Sequential([
    layers.Flatten(input_shape=(28, 28)),  # Входной слой
    layers.Dense(128, activation='relu'),  # Скрытый слой
    layers.Dense(10, activation='softmax') # Выходной слой
])

# Компилируем модель
model.compile(optimizer='adam',
              loss='sparse_categorical_crossentropy',
              metrics=['accuracy'])

# Обучаем модель
model.fit(train_images, train_labels, epochs=5)
